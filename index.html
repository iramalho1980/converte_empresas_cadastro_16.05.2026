Como deseja inserir os CNPJ's para importação dos cadastros de Empresas | do portal RFB?

📋 COPIAR | COLAR CNPJ

Use esta opção se você já tem uma lista de CNPJs para copiar. Você poderá colar os números diretamente no campo de texto.

📊 IMPORTAR | CNPJ - PLANILHA EXCEL

Selecione um arquivo Excel (.xls, .xlsx, .xlsm). O sistema permitirá que você escolha qual coluna contém os CNPJs para inserir no campo para importação do site RFB.

📄 IMPORTAR | CNPJ - TXT

Importe uma lista de CNPJs a partir de um arquivo de texto simples (.txt). O sistema vai identificar apenas o que for CNPJ.
